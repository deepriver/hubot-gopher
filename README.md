Hubot-Gopher
----------------

![](https://raw.githubusercontent.com/deepriver/hubot-gopher/master/gopher_images/30.png)


## Installation

### Add package.json
```
"dependencies": {
  "hubot-gopher": "0.1.0"
}
```

### Add external-scripts.json

```
cat external-scripts.json
[
 "hubot-diagnostics",
 "hubot-help",
 "hubot-heroku-keepalive",
 "hubot-google-images",
 "hubot-google-translate",
 "hubot-pugme",
 "hubot-maps",
 "hubot-redis-brain",
 "hubot-rules",
 "hubot-shipit",
 "hubot-gopher" <<<=
]
```
